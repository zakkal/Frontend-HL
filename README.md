# HL Sales — Frontend

Aplikasi manajemen penjualan & piutang untuk bisnis HL. Dibangun dengan React + Vite + TailwindCSS.

## Fitur
- Dashboard dengan analitik keuangan real-time
- Manajemen pelanggan, produk, dan transaksi
- Sistem piutang & pelunasan dengan input tanggal
- Laporan rekap bulanan (PDF export)
- AI Assistant untuk analisis bisnis
- Deteksi risiko piutang & prediksi pembayaran
- Notifikasi sesi expired otomatis

## Tech Stack
- React 19 + TypeScript
- Vite 8
- TailwindCSS v4
- React Router v7

## Setup Lokal

```bash
npm install
cp .env.example .env
# Isi VITE_API_URL dengan URL backend
npm run dev
```

## Environment Variables

```
VITE_API_URL=https://your-backend.up.railway.app
```
